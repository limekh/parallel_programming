# Лабораторная №1. Перемножение матриц
## Отчёт
### Описание файлов 
- `main.cpp` - исходный код программы на C++. Реализует чтение матриц из файла, запись результата и измерение времени выполнения.
- `verify.py` - скрипт на языке Python для автоматической проверки результатов. Сравнивает результат C++ программы с расчётом через NumPy
- `compile.bat` - компилирует `main.cpp` в `matrix_mult.exe`
- `test.bat` - запускает тесты для разных размеров матриц (32, 64, 128, 256 и 512) и собирает статистику в файл `stats.csv`
- `input.txt` - входной файл с двумя матрицами, автоматически генерируемый при запуске тестов
- `ouyput.txt` - выходной файл с результатом умножения. Содержит матрицу-результат, время выполнения и объём вычислений
- `stats.csv` - файл со статистикой. Содержит данные для матриц разных размеров

### Запуск программы
#### Требования
- MinGW (g++) с поддержкой C++11
- Python 3.x с установленным пакетом `numpy`
#### Зависимости
```cmd
# Проверка g++
g++ --version
# Проверка Python
python --version
# Установка NumPy
pip install numpy
```
#### Компиляция
```cmd
compile.bat
```
#### Запуск тестов
```cmd
test.bat
```

### Процесс проверки
1. C++ программа умножает матрицы и сохраняет результат в output.txt
2. verify.py читает входные матрицы из input.txt
3. NumPy вычисляет результат (A × B)
4. Сравнение результата C++ с результатом NumPy
5. Вывод максимальной погрешности и статуса проверки (PASS/FAIL)
